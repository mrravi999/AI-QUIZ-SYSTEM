# AI-QUIZ-SYSTEM
Project Overview

AI Smart Quiz System is a quiz application developed using Java and web technologies. The project allows users to participate in quizzes, answer multiple-choice questions, and view scores through a leaderboard system.

The system includes:

User Login & Registration
Quiz Interface
Score Calculation
Leaderboard Management
Database Connectivity
Technologies Used
Backend
Java
JDBC
MySQL
Frontend
HTML
CSS
JavaScript
IDE & Tools
IntelliJ IDEA
MySQL Connector/J
Features
User authentication system
Interactive quiz interface
Multiple-choice questions
Automatic score calculation
Leaderboard display
Database integration
Simple and user-friendly UI
